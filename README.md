# Backend II - Entrega (Sessions + JWT)

Servidor backend desarrollado con Node.js, Express y MongoDB que implementa autenticación JWT con cookies httpOnly, patrón Repository, DTOs, middleware de autorización por roles, sistema de recuperación de contraseña con expiración y lógica completa de compra con generación de tickets.

El proyecto consolida buenas prácticas de arquitectura y seguridad aplicadas durante el curso.
